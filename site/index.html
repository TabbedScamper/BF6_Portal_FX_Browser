How these previews are built

Every card is a Portal-placeable FX from the Battlefield 6 Portal editor (the spatial FX_ spawn list). The previews are not videos — they are re-simulated live in your browser from parameters decoded out of the game files: spawn rates, particle lifetimes and sizes, spawn speeds, drag / wind / gravity constants, the authored Color0/Color1 tints (this is what makes the colored marker smokes red/green/violet/yellow), opacity-over-life envelopes, and the exact flipbook sprite sheets with their authored grid layouts.

LIVE effects use the game's billboard/flipbook pipeline and real sprite atlases. PROC effects (sparks) are procedural streak approximations of the game's GPU spark sims. MESH effects spawn the actual game mesh (darts, rockets, debris chunks) with the decoded emitter parameters. SOON effects run on baked compute shaders (screen effects, decals) or live in map bundles not mined yet — those get in-game captures later.

Effects that reference an in-game sound config show a ♪ speaker button — it streams the matching clip from the companion SFX Library (nothing is duplicated here; unmatched configs are listed by name).

Viewer controls: drag = orbit · scroll = zoom · middle mouse = pan · hold right mouse = freelook with WASD fly (Q/E down/up, Shift = boost, scroll = fly speed) — the same scheme as the Model Library site.

The browser player is an approximation: exact particle motion is baked into compiled GPU shaders, so remaining unknowns use documented defaults. Colors, sizes, rates, lifetimes, envelopes, sprites and grids are authentic game data.
